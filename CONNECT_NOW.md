# Connect GitHub to Cloudflare Pages - START NOW

## Step 1: Go to Cloudflare Dashboard

**Open this link in your browser:**
```
https://dash.cloudflare.com
```

---

## Step 2: Create New Pages Project

1. Click **Workers & Pages** (left sidebar)
2. Click **Pages** tab
3. Click **Create application** button
4. Click **Connect to Git**

---

## Step 3: Authorize GitHub

1. Click **GitHub**
2. Click **Authorize Cloudflare**
3. Log in to GitHub if prompted
4. Click **Authorize cloudflare** button

---

## Step 4: Select Your Repository

1. Look for: **Writer-Website-Landing-Page--1-**
2. Click to select it
3. Click **Begin setup**

---

## Step 5: Configure Build Settings

Leave everything as default:
- **Production branch**: `main`
- **Build command**: `npm run build`
- **Build output directory**: `dist`

Click **Save and Deploy**

---

## Step 6: Wait for Deployment

⏳ **Wait 2-3 minutes** for first deployment to complete

You'll see:
- Building... 🔨
- Deploying... 📦
- Success! ✅

---

## Step 7: Get Your URL

Once deployed, you'll see:
```
https://author-fatima-76r-eis.pages.dev
```

**Copy this URL and test it!**

---

## What Happens Next

✅ Your site is now live
✅ Every push to GitHub auto-deploys
✅ Changes appear in 2-3 minutes
