## 3 Simple Steps

### STEP 1: Connect GitHub (5 min)
```
1. Go to https://dash.cloudflare.com
2. Workers & Pages → Pages → Create application
3. Connect to Git → GitHub
4. Select: Writer-Website-Landing-Page--1-
5. Build: npm run build
6. Output: dist
7. Deploy
8. Wait 2-3 minutes ✅
```

### STEP 2: Get Domain (5 min)
```
1. Go to https://www.namecheap.com
2. Buy domain (e.g., authorfatima.com)
3. Note domain name ✅
```

### STEP 3: Connect Domain (5 min)
```
1. Cloudflare: Custom domains → Add domain
2. Copy nameservers
3. Namecheap: Update nameservers
4. Wait 24-48 hours ✅
```
